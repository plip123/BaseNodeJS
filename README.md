# BaseNodeJS

## Install

This repository requires some knowledge of:

- [npm](https://docs.npmjs.com/getting-started)
- [TypeScript](https://www.typescriptlang.org/)

1. Install dependencies:

```bash
$ npm i
```
or

```bash
$ yarn
```

## Usage

Remember to configure your env vars

### Locally

```bash
npm run dev
```
or

```bash
yarn dev
```
